# Vectora Server

Минимальный HTTP-сервер на Go для backend-части проекта Vectora.

## Запуск

```bash
go run ./cmd/server
```

По умолчанию сервер стартует на порту `8080`.

Можно переопределить порт через переменную окружения:

```bash
PORT=3000 go run ./cmd/server
```

## Эндпоинты

- `GET /healthz` — проверка доступности сервиса.
- `GET /` — базовый JSON-ответ сервиса.
